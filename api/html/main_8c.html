Calculate arithmetic progressions via Pythagorean triples.

To avoid the initialization of mpz_t variables within the function, you have to pass initialized mpz_t variables via parameters that then can be used by the function.

Note that the middle value of any arithmetic progression found would be (p1 * p2)^2.
